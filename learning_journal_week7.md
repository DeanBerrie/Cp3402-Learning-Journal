### Estimated Hours

### Content Insights
- This week strengthened my understanding of how PHP integrates with HTML to create dynamic web pages. Using echo to output HTML showed how server-side scripting can control what is displayed to the user, rather than relying on static content.
- Implementing conditional statements (if/else) helped me understand how logic is applied to control program flow, allowing the page to respond differently based on conditions or input.
- Working with loops (for, while, and foreach) highlighted how repetition can automate tasks such as displaying lists of data. Using arrays with foreach was particularly useful for handling structured data efficiently.
- Creating functions with parameters showed me how to make code reusable and more organised. Instead of repeating the same logic, I could encapsulate it into a function and call it when needed, improving readability and maintainability.
- Using include to separate the header and footer across multiple pages introduced me to modular design in web development. This approach reduces duplication and makes it easier to update shared components across a site.
### Learning Insights
- This week reinforced the importance of writing structured and maintainable code. Breaking problems into smaller parts using functions and includes made the overall program easier to understand and manage, especially as complexity increased.
- I found that applying multiple concepts together (such as combining loops, conditions, and functions) was more challenging than learning them individually, but it significantly improved my understanding. It required me to think more logically about how different parts of the code interact.
- Debugging played a key role in my learning. When the code didn’t work as expected, I had to trace through each step to identify issues, which helped me better understand program flow and common mistakes.
- I also realised the importance of planning before coding. Structuring the layout (header, footer, main content) and thinking about how data would be processed made the implementation smoother and reduced the need for major changes later.
- Overall, this week helped shift my thinking from writing simple scripts to developing more structured, reusable, and scalable code, which is essential for building larger web applications.
